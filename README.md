# 💬 YoChat! — Real-Time Chat App with Image Sharing

**YoChat!** is a modern **MERN stack** chat application with **real-time messaging** powered by **Socket.io** and **image sharing** via **Cloudinary**.  
The app includes a responsive UI built with **React**, **Tailwind CSS**, and **DaisyUI**, plus secure authentication using **JWT**.

---

## 🚀 Features

- 🔐 **User Authentication** (JWT-based login & registration)
- 💬 **Real-Time Messaging** with Socket.io
- 🖼 **Image Sharing** in chat (Cloudinary storage)
- 📱 **Responsive Design** for mobile & desktop
- 🔔 Instant message updates without page refresh
- ⚡ Fast and lightweight with **Vite** + React
- 🎨 Modern UI with **Tailwind CSS** + **DaisyUI**
- 🍞 Toast notifications with **react-hot-toast**

---

## 🧱 Tech Stack

### Frontend
- React 18 + Vite
- Tailwind CSS + DaisyUI
- Zustand (state management)
- Socket.io Client
- React Router DOM
- Axios
- Lucide-react icons
- react-hot-toast

### Backend
- Node.js / Express.js
- MongoDB + Mongoose
- JWT for authentication
- Bcrypt.js for password hashing
- Socket.io for real-time communication
- Cloudinary for image hosting
- Cookie-parser for handling cookies


- CORS for cross-origin requests
- dotenv for environment configuration
